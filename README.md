# One District One Product (ODOP) Initiative Website

![Website](https://img.shields.io/badge/Website-Live-brightgreen) ![License](https://img.shields.io/badge/License-MIT-blue) ![HTML](https://img.shields.io/badge/HTML5-orange) ![CSS](https://img.shields.io/badge/CSS3-blue) ![JavaScript](https://img.shields.io/badge/JavaScript-yellow)

***[DEPLOYED LINK](https://onedistrict.netlify.app/)***

The **One District One Product (ODOP)** initiative website was developed to promote and support the local artisans of Jammu and Kashmir. The website showcases unique crafts and products, fostering economic growth and cultural preservation while providing an engaging digital platform for users.

---

## Features
- Showcases unique products from different districts of Jammu and Kashmir  
- High-quality images and detailed product descriptions  
- Interactive elements to enhance user engagement  
- Responsive and user-friendly design for both desktop and mobile devices  
- Promotes cultural heritage while supporting local artisans  

---

## Project Structure
```One-District-One-Product-Website/
├── images/ # Contains all product and background images
├── index.html # Main HTML file
├── styles.css # CSS styles for the website
├── script.js # JavaScript for interactive functionality
├── README.md # Project documentation
```

---

## Installation
Clone the repository:

```
git clone https://github.com/NishuKaushik/One-District-One-Product-Website.git
cd One-District-One-Product-Website
Open index.html in your browser to view the website locally.
```
## Usage

1.Launch the website via the deployed link or locally.

2.Explore products from different districts of Jammu and Kashmir.

3.Click on products for detailed descriptions and images.

4.Navigate the interactive map to explore regional specialties.

## Challenges and Solutions

Authentic Content Sourcing: Collaborated with local artisans and organizations to gather accurate product information.

Design Consistency: Ensured uniform styling across all pages and sections using CSS.

Cross-Browser Compatibility: Tested and optimized the website for multiple browsers.

Responsive Design: Used flexible layouts and media queries to ensure the website works on both desktop and mobile devices.
